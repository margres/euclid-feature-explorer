Euclid Feature-Space Explorer
=============================
Double-click index.html to open it in your web browser.
(Or use "Open me.command" on macOS / "Open me.bat" on Windows.)

Everything runs locally in your browser. No internet connection,
no installation, and no account are required.

Controls:
  - Drag        : pan
  - Mouse wheel : zoom toward the cursor
  - Hover       : show object ID + grade
  - Click       : show the galaxy cutout (graded lenses get a colored border)
  - "Hide background" leaves only the graded lens candidates (A/B/C)
  - "Reset view" returns to the full manifold
